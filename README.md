# Bangladesh Ramadan Digital Calendar (PWA)

A fully functional Progressive Web App (PWA) for Bangladesh that provides accurate Sehri, Iftar, and Prayer times for all 64 districts.

## Features
- **District-based Timing:** Supports all 64 districts of Bangladesh.
- **Real-time Countdown:** Live countdown to Iftar and Sehri.
- **5 Daily Prayer Times:** Accurate timings using the Aladhan API.
- **Ramadan Calendar:** Full monthly view of Ramadan timings.
- **Dua Section:** Sehri Niyyah and Iftar Dua with Arabic, Bangla, and English transliteration.
- **PWA Support:** Installable on mobile and works offline.
- **Dark Mode:** Easy toggle for night-time viewing.
- **Share:** Share timings via WhatsApp or copy the link.

## Tech Stack
- **Frontend:** Pure HTML5, CSS3 (Responsive/Mobile-first), Vanilla JavaScript.
- **API:** Aladhan Prayer Times API.
- **PWA:** Service Workers and Web App Manifest.

## How to Run
1. Extract the `ramadan-pwa.zip` file.
2. Open `index.html` in any modern web browser.
3. For PWA features (Install, Offline mode), serve the folder using a local server (e.g., Live Server in VS Code) or deploy to GitHub Pages.

## Deployment
This project is compatible with GitHub Pages. Simply upload the contents of the `ramadan-pwa` folder to a GitHub repository and enable GitHub Pages in the settings.
